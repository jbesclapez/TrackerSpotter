Quick Start

1. Copy tracker URL for your setup:

HTTP (local): http://127.0.0.1:6969/announce

HTTP (Docker): http://host.docker.internal:6969/announce

UDP (faster): udp://127.0.0.1:6969/announce

UDP (Docker): udp://host.docker.internal:6969/announce

HTTP (IPv6): http://[::1]:6969/announce

UDP (IPv6): udp://[::1]:6969/announce

2. Add it to your torrent client's tracker list

3. Start a torrent and watch events appear below!

Tip: UDP is faster and preferred by most clients
